Como usar o painel — Guia Rápido

Funciona no telemóvel e no computador

1️⃣
Entrar no painel
Abre o link do painel no teu telemóvel.
Escreve a palavra-passe e clica Entrar.
📷
Trocar uma foto
Desce até ao projeto.
Toca em 📷 Trocar na foto que queres mudar.
Escolhe a nova foto da galeria do telemóvel.
Aguarda a confirmação ✅
✏️
Mudar o nome do projeto
Encontra o projeto na lista.
Toca no botão ✏️ ao lado do nome.
Escreve o novo nome.
Toca em 💾 Guardar.
🏷️
Mudar o subtítulo
Abre o projeto (toca no nome).
No fundo aparece 🏷️ Subtítulo.
Edita o texto e toca 💾.
🗑️
Apagar uma foto
Abre o projeto.
Toca em 🗑️ na foto a apagar.
Confirma na janela que aparece.
⚠️ Esta ação não pode ser desfeita.
⏱️
Quando aparece no site?
Após qualquer alteração o site atualiza automaticamente em ~60 segundos.
Não precisas de fazer mais nada!
💡 Dica: Guarda o link do painel nos favoritos do telemóvel para acesso rápido. A palavra-passe fica guardada no browser — só tens de a introduzir uma vez.

Configuração única — 2 minutos

Para enviar fotos, precisas de um código secreto do GitHub (só fazes isto uma vez):

  1. Abre github.com e faz login com a tua conta
  2. Clica na tua foto → Settings → Developer settings
  3. Clica em Personal access tokens → Tokens (classic)
  4. Clica Generate new token → dá um nome → marca ✅ repo
  5. Copia o código gerado e cola aqui em baixo
